================================================================================
                         HOW TO USE
================================================================================

LOAD A SCENARIO:
  1. Click "Scenario ▼" button
  2. Select "Load Scenario"
  3. Choose from:
     - Saved Scenarios tab (your saved files)
     - Sample Scenarios tab (pre-built examples)
  4. Click "Load" or double-click scenario
  5. Map automatically populates with objects and routes

SAVE A SCENARIO:
  1. Set up objects and routes on map
  2. Click "Scenario ▼" button
  3. Select "Save Scenario"
  4. Enter scenario name and description
  5. Click "Save"
  6. File saved to: AppData/VistarPlanner/scenarios/

================================================================================
                       FILE FORMAT
================================================================================

FORMAT: JSON (.json extension)
LOCATION: AppData/VistarPlanner/scenarios/
STRUCTURE:
  {
    "name": "Scenario Name",
    "description": "Description",
    "createdDate": "2024-12-26T10:00:00",
    "objects": [ {id, type, lat, lon, alt, data} ],
    "routes": [ {id, name, waypoints, data} ],
    "metadata": { }
  }
