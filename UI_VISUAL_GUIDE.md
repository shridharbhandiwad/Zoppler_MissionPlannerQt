## Keyboard Shortcuts

While dialogs don't currently implement keyboard shortcuts, these could be added in the future:

- **Ctrl+L**: Open Load Scenario dialog
- **Ctrl+S**: Open Save Scenario dialog
- **Enter**: Confirm action (Load or Save)
- **Esc**: Cancel dialog
- **Tab**: Navigate between fields

## Accessibility Features

- **Tab Order**: Logical tab order through form fields
- **Focus Indicators**: Clear visual focus indicators
- **Tooltips**: Descriptive tooltips on all buttons
- **Labels**: All input fields have associated labels
- **Error Messages**: Clear, descriptive error messages
- **Color Contrast**: High contrast text for readability
